# 🇳🇵 KidzPatshala — Kids Learning Platform

**KidzPatshala** (किड्ज पाठशाला) is an open educational platform for Nepali-American kids to learn the Nepali language, culture, and more — right from their phones.

> *Patshala* (पाठशाला) means "school" in Nepali. KidzPatshala is a school in your pocket!

## 🌟 Nepali Sikau — Feature Highlights

- **📚 Learn** — 9 categories: Alphabet (43 letters), Numbers, Phrases, Animals, Colors, Body, Food, Nature, Family
- **🎯 Quiz** — 4 modes: Multiple choice, Type it in, Listen & pick, Emoji match
- **✏️ Trace** — Finger-trace Devanagari letters on screen
- **🎬 Videos** — 12 curated Nepali kids YouTube videos
- **🏆 Badges** — 12 badges and a 5-star progress system
- **🔊 Sound** — Nepali text-to-speech on every word
- **📲 PWA** — Installable on iPhone, Android & Samsung with offline support

## 🚀 Install on Your Phone (No App Store Needed!)

### iPhone
1. Open the app link in **Safari**
2. Tap the **Share** button (box with arrow)
3. Tap **"Add to Home Screen"**
4. Tap **Add** — done! 🎉

### Android / Samsung
1. Open the app link in **Chrome**
2. Tap the **three dots menu** (⋮)
3. Tap **"Add to Home Screen"** or **"Install App"**
4. Tap **Install** — done! 🎉

## 🛠 Tech Stack

- Pure HTML, CSS, JavaScript — no frameworks, no build step
- Progressive Web App (PWA) with Service Worker for offline use
- Web Speech API for Nepali text-to-speech
- YouTube IFrame API for video playback
